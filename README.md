# YasDex

In the wake of veekun and marriland search not being updated for newer gens, a new online dex with search functionality
arises to fill the void. [Hosted via Github pages.](https://davidy22.github.io/YasDex/)

All data courtesy of https://github.com/PokeAPI.

## Known issues
The checkboxes for branching evolutions don't give 100% correct information due to the data source not fully
